    Batch Ship Packages

    Use this API to batch ship packages from a single shop by providing multiple package IDs in the request body. You can retrieve package IDs by running the Get Order Detail API.

    This API is available for both TikTok shipping orders and seller shipping orders.

    • TikTok Shipping: Schedule a package handover time for TikTok shipping carriers to pick up a package from the seller. Include the pickup start_time and end_time.
    • Seller Shipping: The seller arranges their own shipping and provides a tracking_number and shipping_provider_id.

    URL

    POST /fulfillment/202309/packages/ship

    Request

    View common parameters.

    Header

    Properties Type Description
    content-type
    Required    		 string Allowed type: application/json

    Query

    Properties Type Description
    shop_cipher Required string Use this property to pass a shop's cipher in the requesting API. Failure to pass a cipher for an authorized cross-border shop will result in an error. You can retrieve a list of ciphers for authorized shops by running the Get Authorized Shops API.

    Body

    Properties Type Description
    packages                                                                            
    Required    		 []object] An array of package objects
        packages.id
      Required    		 string The package ID
        packages.handover_method string The package handover type. Available values are
    • PICKUP: A shipping provider will pick up the package(s) from the seller's pickup address.
    • DROP_OFF: The seller will drop off the package(s) at a designated location.
        packages.pickup_slot []object The package pickup timeslot when handover_method is PICKUP.
            pickup_slot.start_time string The start date and time of the package pickup time slot. This value is a Unix time stamp.
            pickup_slot.end_time string The end date and time of the package pickup time slot. This value is a Unix time stamp.
        packages.self_shipment []object This object is only needed when the seller is shipping packages.
    Review the shipping_type field in the Get Package Detail API response to determine who is shipping.
    Use the shipping provider ID retrieved from the Get Shipping Providers API and upload the corresponding tracking number.
            self_shipment.tracking_number
        Required    		 string The package's tracking number when shipping_type is SELLER.
            self_shipment.shipping_provider_id
        Required    		 string The shipping provider ID when shipping_type is SELLER. Use the Get Shipping Providers API to retrieve this value.
